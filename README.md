# StayFresh

## Overview

StayFresh is a fitness and nutrition web application built using the MERN stack. The application helps users achieve their fitness goals by generating meal recommendations based on their body information, activity level, and fitness goals.

## Features

* User Registration
* User Login
* Email Verification
* Secure Password Hashing
* User Profile Setup
* Daily Calorie Recommendations
* Breakfast, Lunch, Dinner, and Snack Suggestions
* Meal Plan Generator
* Meal Plan Regeneration
* Responsive Design
* Dashboard with User Information and Nutrition Recommendations
